# kakaobotMeal
카카오톡 봇으로 급식을 검색해주는 기능입니다.

### Description
- 라이노엔진을 이용하여 자바스크립트로 코딩했습니다
- '/급식검색 ○○학교/(일)' 명령어로 급식을 검색합니다


### 사용법
- 카카오톡에 봇을 적용시켜줄 수 있는 어플을 다운합니다
- 해당 소스를 적용하고 컴파일 시킵니다
